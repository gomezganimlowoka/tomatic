The Tomato Leaf Disease Detector is an AI-powered web application designed to help farmers and gardeners identify common diseases affecting tomato plants by analyzing leaf images.

Using a trained deep learning model, the application predicts the type of disease present on the leaf and provides information about the cause and prevention methods to help users take timely action.

This tool aims to improve crop health and yield by enabling early detection and management of tomato leaf diseases.
